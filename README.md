##  Overview
A comprehensive repository for **Introduction to Data Science** featuring:
- **AI-powered data science agents** built in Google Colab for automating data tasks (upload, preview, analysis, visualization).
- **Python fundamentals study materials** as organized Jupyter notebooks.

>  Perfect for students learning **Python basics + data science automation + agent-based systems**.

---

##  Features

### Agents:
-  Instant CSV file upload & preview

### Study Materials:
- Structured Python tutorials from basics to data structures
- Hands-on Jupyter notebooks
- Progressive learning path

##  Study Materials

Complete **Python Fundamentals** course in Jupyter notebooks:

|  Notebook | Topics Covered |
|-------------|----------------|
| [Python Basics: Data Types & Variables](Study_Materials/Python_Basics_DataTypes_Variables.ipynb) | Print/output, data types, variables, type conversion, user input, literals |
| [Operators, Conditionals & Loops](Study_Materials/Python_Operators_Conditionals_Loops.ipynb) | Arithmetic/relational operators, if-else, modules, while/for loops |
| [Advanced Loops & Patterns](Study_Materials/Python_AdvancedLoops_Patterns.ipynb) | Nested loops, number patterns, break/continue/pass |

| [Lists](Study_Materials/Python_Lists.ipynb) | List creation, operations, functions, comprehensions |
| [Dictionaries](Study_Materials/Python_Dictionaries.ipynb) | Dict creation, operations, functions, comprehensions |

**Run notebooks:** Open in Jupyter/Colab for interactive learning.

##  Quick Start

### For Agents (Google Colab):
1. Open the agent MD file (e.g., [CSV Upload Agent](Agents/csv-upload-agent/csv-upload-agent.md))
2. Copy-paste code cells into new Colab notebook
3. Run commands to interact with the agent

### For Study Materials:
1. Download notebooks from `Study_Materials/`
2. Open in Jupyter Lab or Google Colab
3. Execute cells sequentially
